## 🎲 二、random：電腦幫你「亂數抽籤」

### 1️⃣ random.randint(a, b)

👉 會產生 **a 到 b 之間的整數（包含 a 和 b）**

例子：

```python
random.randint(1, 5)
```

意思是：
➡️ 電腦隨機選 **1、2、3、4、5** 其中一個

---

### 2️⃣ random.randrange()

👉 也是產生亂數，但可以控制「從哪到哪、隔幾個」

```python
random.randrange(6)        # 0～5
random.randrange(1, 11)    # 1～10
random.randrange(0, 10, 2) # 0,2,4,6,8
```

🧠 小技巧：

* 最後一個數字 **不會被選到**

---

### 3️⃣ 統計出現次數

電腦可以玩 1000 次，算看看每個數字出現幾次
➡️ 就像在算「抽籤統計表」

## 🔁 三、迴圈：叫電腦重複做事情

### 1️⃣ for 迴圈（固定次數）

```python
for i in range(5):
    print(i)
```

➡️ 會印出：0 1 2 3 4

---

### 2️⃣ while 迴圈（做到條件不成立）

```python
i = 0
while i < 5:
    print(i)
    i += 1
```

---

### 3️⃣ break（強制停止）

```python
if i == 3:
    break
```

➡️ 走到 3 就停下來！

## ❓ 四、猜數字遊戲（超重要）

### 遊戲規則：

* 電腦先想一個數字（0～100）
* 玩家一直猜
* 電腦會說：

  * 太小了
  * 太大了
  * 猜對了 🎉

### 重點學到：

* while 迴圈
* if / elif / else 判斷
* 記住「最小值、最大值」

## 🗂️ 五、字典（dictionary）：像「成績表」

### 1️⃣ 字典長這樣

```python
scores = {"國文": 90, "數學": 80}
```

🧠 想成：

* 「國文」是標題
* 90 是分數

---

### 2️⃣ 讀取資料

```python
scores["國文"]   # 90
```

---

### 3️⃣ 新增、修改

```python
scores["英文"] = 85   # 新增
scores["國文"] = 95   # 修改
```

---

### 4️⃣ 刪除

```python
scores.pop("國文")
```

---

### 5️⃣ 看裡面有什麼

```python
scores.keys()    # 科目
scores.values()  # 分數
scores.items()   # 全部
```

---

### 6️⃣ 算平均分數

```python
total / 次數
```

➡️ 把分數加起來再除以幾次

## ➕ 六、快速加減法（懶人寫法）

```python
a += b   # a = a + b
a -= b   # a = a - b
a *= b   # a = a * b
a /= b   # a = a / b
```

🧠 就是「快速版寫法」
